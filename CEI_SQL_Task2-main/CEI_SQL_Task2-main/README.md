# CEI_SQL_Task2

As a CEI intern, this is my Week 2 SQL assignment focused on designing and analyzing an E-Commerce Sales Database using SQL. The project covers database creation, constraints, filtering, aggregation, joins, indexing concepts, transactions, and ACID properties to generate meaningful business insights.

In this assignment, I worked with a relational database consisting of Customers, Products, Orders, and Order Items tables. Various SQL operations were performed to analyze customer behavior, product performance, and sales trends.
